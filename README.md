# 🚀 Task 2 - Full Stack User Registration System

This project is developed as part of my Full Stack Development Internship.

It is a complete user registration system built using React (Frontend) and Node.js with Express (Backend).

---

## 🔥 Features

- ✅ User Registration Form
- ✅ Client-side Validation (React)
- ✅ Server-side Validation (Node.js)
- ✅ Password Strength Indicator (Weak / Medium / Strong)
- ✅ Responsive & Professional UI (CSS + Bootstrap)
- ✅ API Integration between Frontend & Backend
- ✅ Temporary Data Storage (Array)

---

## ⚠️ Validation Features

- Shows warning if fields are empty
- Validates mobile number (must be 10 digits)
- Ensures password length (minimum 6 characters)
- Displays real-time validation messages
- Password strength indicator for better UX

---

## 🛠️ Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Styling: CSS, Bootstrap
- API Handling: Axios
